Gizlilik Politikası

Bu gizlilik politikası, web sitemizi ziyaret eden kullanıcılarımızın gizliliğini korumak amacıyla oluşturulmuştur.

Toplanan Bilgiler

Web sitemizi ziyaret ederken, belirli bilgiler otomatik olarak kaydedilebilir. Bu bilgiler arasında IP adresi, tarayıcı türü, ziyaret tarihi ve saati, sayfalara erişim sırasında kullanılan bağlantılar ve diğer istatistiksel veriler yer alabilir.

Çerezler

Web sitemiz, kullanıcı deneyimini geliştirmek ve tercihlerinizi hatırlamak için çerezler kullanabilir. Çerezler, tarayıcınız tarafından yerel olarak depolanan küçük metin dosyalarıdır.

Veri Kullanımı

Toplanan bilgiler, web sitemizin işleyişiyle ilgili istatistiksel analizler yapmak, içerik ve hizmetlerimizi geliştirmek, kullanıcı deneyimini kişiselleştirmek ve reklamları hedeflemek amacıyla kullanılabilir.

Veri Paylaşımı

Toplanan bilgiler, yasal gereklilikler veya yetkili makamlar tarafından talep edildiği durumlar haricinde üçüncü taraflarla paylaşılmaz.

Veri Güvenliği

Web sitemizde toplanan bilgilerin güvenliği için uygun önlemler alınmaktadır. Ancak, internet üzerinden iletilen verilerin tam güvenliğini garanti edemeyiz.

Diğer Web Siteleri

Web sitemiz, farklı web sitelerine bağlantılar içerebilir. Bu web sitelerinin gizlilik uygulamalarından sorumlu değiliz. Dış bağlantıları kullanırken ilgili web sitelerinin gizlilik politikalarını okumanızı öneririz.

Gizlilik Politikası Güncellemeleri

Gizlilik politikamızı zaman zaman güncelleyebiliriz. Bu sayfayı düzenli olarak kontrol etmenizi öneririz.

İletişim

Gizlilik politikamızla ilgili herhangi bir sorunuz varsa, iletişim kısmından ulaşın.
